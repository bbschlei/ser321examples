You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /calculateDistance?x1=0&y1=0&x2=3&y2=4 -- Calculates the distance between two points (Example: 0,0 to 3,4)
  • /analyzeText?text=hello%20world%20this%20is%20a%20test -- Analyzes the given text to find the number of words, the longest and shortest word

File Structure in www (you can use /file/www/FILENAME):

${links}

Custom Requests

Calculate Distance: This function calculates the Euclidean distance between two points provided via URL parameters x1, y1 (for the first point) and x2, y2 (for the second point).

Example: http://localhost:9000/calculateDistance?x1=0&y1=0&x2=3&y2=4

Analyze Text: Analyze provided text to find the number of words, the longest and shortest word. Ensure text is URL encoded.

Example: http://localhost:9000/analyzeText?text=hello%20world%20this%20is%20a%20test
